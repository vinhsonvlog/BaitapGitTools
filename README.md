# BaitapGitTools

## Commerce UI Template

This repository now includes a responsive storefront UI template:

- `index.html`: layout and sections (hero, products, promo banner)
- `styles.css`: visual theme, responsive grid, and entry animations
- `script.js`: product filter chips and flash-sale countdown

## Run

1. Open `index.html` in your browser.
2. (Optional) Use a Live Server extension in VS Code for auto reload.

## Notes

- Mobile and desktop layouts are both supported.
- Product cards can be filtered by category.
- You can replace static product data with API data later.
